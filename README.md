# D.E.T Trade Licence Authenticator

Static website for the D.E.T Trade Licence Authenticator.

## Pages

- `index.html` - Licence verification page
- `privacy.html` - Privacy policy
- `terms.html` - Terms and disclaimer
- `contact.html` - Contact support
- `det-logo.svg` - Site logo
- `favicon.svg` - Browser icon

## Deployment

This site can be deployed through GitHub Pages, Cloudflare Pages, Netlify, Vercel, or any static web host. No build command is required.

For Cloudflare Pages, use:

- Build command: none
- Output directory: `/`
